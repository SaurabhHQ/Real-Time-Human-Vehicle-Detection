# Real-Time-Human-Vehicle-Detection
Computer Vision project that detects humans and vehicles in real-time video streams using YOLO, OpenCV, and Python with high accuracy and fast inference.
🚗 Human & Vehicle Danger Detection System

 📌 Project Overview

The Human & Vehicle Danger Detection System is an AI-powered computer vision project that detects humans and vehicles in real-time video streams using YOLOv8 and OpenCV. The system analyzes object positions and movement patterns to identify potentially dangerous situations where a moving vehicle approaches a nearby pedestrian. When a risk is detected, the system generates a visual danger alert, helping improve road safety and surveillance monitoring.



 ✨ Features

* Real-time human detection
* Real-time vehicle detection
* YOLOv8-based object recognition
* Vehicle movement analysis
* Human-vehicle proximity monitoring
* Automatic danger alert generation
* Visual warning system with bounding boxes
* Video stream processing using OpenCV
* Lightweight and efficient implementation



 🛠 Technologies Used

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* NumPy
* Computer Vision
* Deep Learning

 ⚙️ Installation Steps

 1. Clone the Repository

```bash
git clone https://github.com/your-username/human-vehicle-detection.git
cd human-vehicle-detection
```

 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

 3. Run the Project

```bash
python main.py
```

 4. Press 'Q' to Exit

The application will start processing the input video and display real-time detection results.
